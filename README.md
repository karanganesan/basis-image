# Basis Image (Custom Element)

like &lt;img> but supports .basis files using Web Component

Tag &lt;basis-img>

# Live

https://basis-image.karan.pro/

# Usage

```html
<script src="https://basis-image.karan.pro/build/bundle.js" />
<basis-img src="image.basis" />
```
